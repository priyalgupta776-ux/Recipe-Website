The TastyTwirl – Recipe Website

The TastyTwirl is  a simple and visually appealing recipe website created using HTML , CSS and JAVASCRIPT.
This project showcases a clean homepage, recipe cards, a recipe search layout, and a dedicated recipe details page. Along with a about us page and a simple from based contact us page.

Features

Home Page

* Attractive site title with a colorful header
* Recipe category buttons
* Recipe cards with images, ratings, and preparation time
* Hover effects that make recipe names pop up
* Clean, modern layout with fully centered content

Recipe Page

* Search bar
* Trending recipes section
* Equal-sized recipe cards
* Clean alignment and spacing
* A “Back to Home” button
* Internal & external CSS styling
  
Image Hover Effects

* Shows recipe names when hovered
* Adds interactivity and improves user experience


Technologies Used

* HTML
* CSS (Internal + External Stylesheets)
* Visual Studio Code (VS Code)


How to Use

1. Clone or download the repository
2. Open the project folder in VS Code
3. Use the “Go Live” extension (or open in browser)
4. Explore the homepage and recipe pages
